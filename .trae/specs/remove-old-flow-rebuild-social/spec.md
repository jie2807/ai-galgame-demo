# 舆论锻造所：移除旧流程，重构为纯社交浏览体验 Spec

## Why
当前进入舆论锻造所后仍展示旧版"创作中心→接商单→发布视频→舆论操控"流程，与"浏览真实社交网络"的核心体验严重不符。需要彻底移除商单系统、抖加推广、舆论操控等旧玩法，将私信改造为Agent聊天，保留投稿功能，让玩家获得沉浸式社交网络浏览体验。

## What Changes
- **BREAKING**: 移除商单系统（generateOrders、acceptOrderFromMessage、商单UI、商单成就）
- **BREAKING**: 移除抖加推广面板（cwDoujiaPanel 及全部相关代码）
- **BREAKING**: 移除舆论操控功能（数据中心中的操控按钮、操控历史）
- **BREAKING**: 移除主视图（cwMainView）作为独立视图，平台视图成为唯一主界面
- **BREAKING**: 重构私信系统——从"商单私信"改为"Agent聊天对话"
- 重构个人空间页面——移除抖加/收益/创作中心Tab，保留私信/关注/视频/成就
- 重构导航栏——移除"创作中心"链接，改为"动态"
- 保留投稿功能——简化为从个人空间或导航栏入口触发的轻量发布对话框
- 清理设置面板——移除商单难度评估、性能模式选项
- 清理评论管理——移除水军筛选

## Impact
- Affected specs: `restructure-cw-ai-agent-social`（旧流程移除部分在此实现）
- Affected code:
  - `index.html`：主视图HTML大幅精简/移除、私信系统重构、个人空间重构、导航栏重构、商单/抖加/操控相关JS全部移除
  - `pages/creator-wars/creator-wars.css`：可能需要少量样式调整

## ADDED Requirements

### Requirement: 平台视图为唯一主界面
进入舆论锻造所后，玩家 SHALL 直接进入平台视图（B站风格首页），不再经过主视图的旧流程。

**视图架构变更**：
- 移除 `cwMainView` 作为独立视图
- `cwSwitchView` 只保留 `platform`、`detail` 两个视图
- "开始浏览"按钮点击后直接进入平台视图
- 平台视图即为玩家的主界面，所有功能通过导航栏、下拉菜单、弹窗访问

**导航栏重构**：
```
左侧：[舆论TV Logo] [首页] [热门] [动态] [🏠]
中间：[搜索框]
右侧：[⚡影响力] [🔔通知] [👤头像▼]
```

- "首页"：当前平台视图（信息流）
- "热门"：按播放量排序的信息流
- "动态"：展示关注Agent的最新动态（发布、直播、评论互动）
- 移除"创作中心"导航链接
- 移除"频道"导航链接（与分类标签重复）

#### Scenario: 进入游戏直接看到首页
- **WHEN** 玩家配置完Agent并点击"开始浏览"
- **THEN** 直接进入B站风格首页信息流
- **AND** 不再看到旧的创作中心/商单界面

#### Scenario: 导航栏切换
- **WHEN** 玩家点击导航栏"动态"
- **THEN** 首页内容切换为关注Agent的最新动态列表

### Requirement: 重构私信为Agent聊天
私信系统 SHALL 从"商单私信"重构为"Agent聊天对话"，玩家通过私信与Agent进行一对一聊天。

**新私信系统设计**：

```
┌──────────────────────────────────────────┐
│  📬 私信                            ✕   │
├──────────────────────────────────────────┤
│  ┌──────┐                                │
│  │ 头像 │  真相调查员              2分钟前│
│  │      │  你好，有什么想讨论的吗？       │
│  └──────┘                                │
│  ┌──────┐                                │
│  │ 头像 │  键盘侠老哥              5分钟前│
│  │      │  呵呵，你那观点也太天真了       │
│  └──────┘                                │
│  ┌──────┐                                │
│  │ 头像 │  热点追踪者             10分钟前│
│  │      │  刚发了新视频，来看看！         │
│  └──────┘                                │
├──────────────────────────────────────────┤
│  💬 输入消息...              [发送]      │
└──────────────────────────────────────────┘
```

**私信列表数据来源**：
- 列表项来自 `cwEngine.agents`，每个Agent一条对话
- 显示Agent头像（portrait优先，fallback emoji）、名称、最后一条消息、时间
- 未读消息显示红点/数字

**点击对话进入聊天**：
- 复用现有 `charchat` 系统打开与Agent的聊天窗口
- Agent ID格式：`agent_${agent.id}`
- 聊天风格由Agent的 `personality` 和 `background` 决定
- 支持联网搜索（已有 `callAIWithWebSearch`）

**私信弹窗改为聊天界面**：
- 左侧：Agent对话列表
- 右侧：与选中Agent的聊天记录（复用charchat渲染）
- 底部：消息输入框 + 发送按钮

**移除内容**：
- "🔄 刷新私信"按钮（不再生成商单）
- "🎨 自由创作"按钮（移到投稿入口）
- 商单类型消息的展示和交互

**新增内容**：
- Agent主动发私信的场景：
  - Agent发布新视频时，可能给关注的玩家发一条通知私信
  - Agent直播开始时，给关注的玩家发直播通知
  - 键盘侠Agent可能主动挑衅玩家

#### Scenario: 打开私信看到Agent对话
- **WHEN** 玩家点击"我的私信"
- **THEN** 显示所有Agent的对话列表
- **AND** 每条显示Agent头像、名称、最后消息摘要

#### Scenario: 与Agent聊天
- **WHEN** 玩家在私信中点击某个Agent
- **THEN** 打开与该Agent的聊天界面
- **AND** 可以发送消息，Agent根据性格回复

#### Scenario: Agent主动发私信
- **WHEN** 某Agent发布新视频
- **THEN** 已关注该Agent的玩家收到一条通知私信
- **AND** 私信列表中该Agent条目显示未读标记

玩家 SHALL 仍能投稿发布内容，但入口简化为轻量对话框。

**投稿入口**：
1. 平台视图导航栏右侧添加"📹 投稿"按钮（在头像左侧）
2. 个人空间"📹 视频"Tab中添加"+ 投稿新视频"按钮

**投稿对话框设计**：
```
┌──────────────────────────────────────────┐
│  📹 投稿                             ✕  │
├──────────────────────────────────────────┤
│  标题                                    │

│  │ 输入视频标题...                  │    │
```

**投稿流程**：
1. 玩家填写标题和内容（可AI辅助生成）
2. 选择分类
3. 点击"发布视频"
4. 显示进度条（模拟发布过程）
5. 发布成功后视频出现在首页信息流中
6. Agent可能对玩家发布的视频进行评论

**与旧流程的区别**：
- 不再需要选择框架/素材
- 不再需要审核流程
- 不再有抖加推广步骤
- 发布后直接进入信息流

#### Scenario: 从导航栏投稿
- **WHEN** 玩家点击导航栏"📹 投稿"按钮
- **THEN** 弹出投稿对话框
- **AND** 填写内容后可发布

#### Scenario: 发布后视频出现在首页
- **WHEN** 玩家发布视频成功
- **THEN** 视频出现在首页信息流中
- **AND** 标记为"我的"视频

## REMOVED Requirements

### Requirement: 商单系统
**Reason**: 核心玩法从"操控舆论完成商单"改为"浏览社交网络"，商单不再需要
**Migration**: 无需迁移，私信改为Agent聊天

### Requirement: 抖加推广
**Reason**: 玩家不再需要推广功能，内容由Agent自然生成和传播
**Migration**: 无需迁移

### Requirement: 舆论操控
**Reason**: 玩家角色从"舆论操控者"变为"社交网络用户"，不再有操控舆论的能力
**Migration**: 舆论指标保留但改为被动展示，由Agent内容自然影响

### Requirement: 主视图（cwMainView）
**Reason**: 主视图承载旧流程（订单/私信/工作室），新流程直接进入平台视图
**Migration**: 订单区域移除；私信弹窗保留但重构为Agent聊天；工作室区域移除

### Requirement: 创作中心面板
**Reason**: 旧创作中心流程已移除，投稿功能简化为对话框
**Migration**: 投稿入口移到导航栏和个人空间

### Requirement: 性能模式选择
**Reason**: Agent系统始终启用，不再需要高低消耗模式切换
**Migration**: Agent活跃度设置替代性能模式
