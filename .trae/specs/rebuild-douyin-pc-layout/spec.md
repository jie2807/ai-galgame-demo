# 抖音PC端布局全面重做 Spec

## Why
当前抖音应用使用的是移动端布局（底部导航栏+全屏视频滑动），完全不符合PC端抖音的实际设计。真实PC端抖音采用**左侧导航栏+中间视频播放区+右侧评论区**的三栏布局，视觉体验和交互逻辑与移动端截然不同。需要全面重做，对标真实PC端抖音。

## What Changes
- **BREAKING**：移除底部导航栏，改为左侧固定导航栏
- **BREAKING**：移除全屏视频滑动模式，改为居中视频播放器+上下切换
- **BREAKING**：评论面板从底部滑出改为右侧固定面板
- **新增**：左侧导航栏——上半部分（首页/推荐/关注/朋友/我的），下半部分（娱乐/知识/二次元/游戏/音乐/美食分类）
- **新增**：中间视频播放区——居中视频+底部用户信息+操作按钮
- **新增**：右侧评论区——固定显示在视频右侧
- **修改**：个人主页改为左侧导航栏内"我的"页面
- **修改**：发布/AI创作面板改为居中弹窗
- **修改**：朋友页和消息页改为左侧导航栏内的子页面

## Impact
- Affected specs: `redesign-douyin-app`、`redesign-douyin-layout`、`build-douyin-ai-anchor-agent`
- Affected code:
  - `index.html`：renderDouyinContent函数全面重写HTML结构
  - `pages/creator-wars/creator-wars.css`：.douyin-*样式全面重写

### 真实PC端抖音布局结构
```
┌──────────────────────────────────────────────────────┐
│ ┌────────┐ ┌──────────────────────┐ ┌──────────────┐ │
│ │ 左侧   │ │     中间视频区       │ │   右侧       │ │
│ │ 导航栏 │ │                      │ │   评论区     │ │
│ │        │ │  ┌──────────────┐    │ │              │ │
│ │ 首页   │ │  │              │    │ │  XX条评论    │ │
│ │ 推荐   │ │  │   视频播放   │    │ │  ──────────  │ │
│ │ 关注   │ │  │   区域       │    │ │  用户A       │ │
│ │ 朋友   │ │  │              │    │ │  评论内容... │ │
│ │ 我的   │ │  └──────────────┘    │ │              │ │
│ │ ────── │ │  @用户名  +关注      │ │  用户B       │ │
│ │ 娱乐   │ │  视频描述文字...     │ │  评论内容... │ │
│ │ 知识   │ │  #标签 #标签         │ │              │ │
│ │ 二次元 │ │  🎵 原声 - xxx      │ │  ──────────  │ │
│ │ 游戏   │ │  ❤ 1.2万 💬 342     │ │  写评论...   │ │
│ │ 音乐   │ │  🔖 856  🔗 126     │ │  [发送]      │ │
│ │ 美食   │ │                      │ │              │ │
│ └────────┘ └──────────────────────┘ └──────────────┘ │
└──────────────────────────────────────────────────────┘
```

### 关键设计特征
1. **左侧导航栏**：固定宽度约72px（图标模式）或200px（展开模式），深色背景
2. **中间视频区**：视频居中显示，下方显示用户信息和操作按钮
3. **右侧评论区**：固定显示，不遮挡视频，约320px宽
4. **无底部导航栏**：所有导航在左侧
5. **视频切换**：上下方向键或滚轮切换，不是左右滑动
6. **评论始终可见**：不需要点击打开评论区

## ADDED Requirements

### Requirement: 左侧导航栏
系统 SHALL 实现PC端风格的左侧固定导航栏，替代当前的底部导航栏。

**导航栏结构**：
```
┌─────────────┐
│  🎵 抖音    │ ← Logo区域
│─────────────│
│  🏠 首页    │ ← 主导航（上半部分）
│  🔥 推荐    │
│  👥 关注    │
│  👫 朋友    │
│  👤 我的    │
│─────────────│
│  🎬 娱乐    │ ← 内容分类（下半部分）
│  📚 知识    │
│  🎮 二次元  │
│  🎯 游戏    │
│  🎵 音乐    │
│  🍜 美食    │
│─────────────│
│  ➕ 发布    │ ← 底部操作
└─────────────┘
```

**导航栏样式**：
- 宽度：72px（图标模式），图标+文字垂直排列
- 背景：#1a1a1a（深灰黑色）
- 当前选中项：左侧3px红色竖条 + 文字变白 + 图标高亮
- 未选中项：文字和图标rgba(255,255,255,0.5)
- hover效果：背景rgba(255,255,255,0.05)
- 顶部Logo区域：抖音音符图标
- 底部发布按钮：红色圆形+号按钮

**导航行为**：
- 点击"首页"：显示推荐视频流
- 点击"推荐"/"关注"：切换视频流类型
- 点击"朋友"：显示朋友动态页
- 点击"我的"：显示个人主页
- 点击内容分类：按分类筛选视频
- 点击"发布"：打开AI创作弹窗

#### Scenario: 导航栏交互
- **WHEN** 用户点击左侧导航栏的"推荐"
- **THEN** 中间区域显示推荐视频流，"推荐"项显示选中状态（左侧红色竖条）
- **AND** 其他导航项取消选中

---

### Requirement: 中间视频播放区
系统 SHALL 实现PC端风格的居中视频播放区，替代当前的全屏滑动模式。

**视频区结构**：
```
┌────────────────────────────────┐
│                                │
│    ┌──────────────────────┐    │
│    │                      │    │
│    │    视频播放区域       │    │
│    │    (渐变背景+Lottie) │    │
│    │                      │    │
│    │    字幕区域           │    │
│    │                      │    │
│    └──────────────────────┘    │
│                                │
│    @用户名  [+ 关注]           │
│    视频描述文字...             │
│    #标签 #标签                 │
│    🎵 原声 - xxx              │
│                                │
│    ❤ 1.2万  💬 342  🔖 856   │
│    🔗 126   🔇 静音           │
│                                │
│    ▲ 上一个  ▼ 下一个          │
└────────────────────────────────┘
```

**视频区样式**：
- 视频播放区域：居中显示，16:9或9:16比例，带圆角和阴影
- 视频下方：用户信息（头像+用户名+关注按钮+描述+标签+音乐）
- 操作按钮：水平排列在用户信息下方（❤💬🔖🔗🔇）
- 上下切换：视频区域两侧或底部有上下箭头按钮
- 背景：深色（#121212）

**视频切换**：
- 点击"下一个"按钮或按↓键切换到下一个视频
- 点击"上一个"按钮或按↑键切换到上一个视频
- 滚轮也可以切换视频
- 切换时有淡入淡出过渡效果（不是滑动）

#### Scenario: 视频播放
- **WHEN** 用户浏览抖音
- **THEN** 视频居中显示在中间区域，右侧显示评论区
- **AND** 视频下方显示用户信息和操作按钮
- **AND** 可以通过箭头按钮或键盘切换视频

### Requirement: 右侧评论区
系统 SHALL 实现PC端风格的右侧固定评论区，替代当前的底部滑出评论面板。

**评论区结构**：
```
┌──────────────────────┐
│  342条评论    最热 ↓ │ ← 标题栏
│──────────────────────│
│  用户A · 2小时前     │
│  太好看了！收藏了     │
│  ❤ 128  回复         │
│──────────────────────│
│  用户B · 3小时前     │
│  这个角度拍得真棒     │
│  ❤ 56   回复         │
│──────────────────────│
│  ...更多评论...      │
│──────────────────────│
│  [写评论...]  [发送] │ ← 输入区
└──────────────────────┘
```

**评论区样式**：
- 宽度：约320px，固定在视频右侧
- 背景：#1e1e1e（深灰色）
- 评论列表可滚动
- 标题栏显示评论数量和排序选项
- 底部固定输入框+发送按钮
- 始终可见，不需要点击打开

**评论交互**：
- 输入评论后点击发送或按Enter
- 评论发送后自动添加到列表顶部
- AI评论功能保留（AIServiceHub）

#### Scenario: 评论区始终可见
- **WHEN** 用户浏览视频
- **THEN** 右侧始终显示当前视频的评论区
- **AND** 切换视频时评论区内容更新
- **AND** 用户可以直接在输入框写评论

系统 SHALL 在中间区域显示个人主页，替代当前的独立全屏页面。

**个人主页结构**：
```
┌────────────────────────────────┐
│  ← 返回    个人主页    ⋮      │
│────────────────────────────────│
│  ┌──────┐  舆论锻造师          │
│  │ 头像 │  抖音号：xxx         │
│  └──────┘  [编辑资料]          │
│                                │
│  256关注  10.2万粉丝  86.5万获赞│
│────────────────────────────────│

│────────────────────────────────│
│  ┌──┐ ┌──┐ ┌──┐              │
│  │  │ │  │ │  │              │
│  └──┘ └──┘ └──┘              │
│  ┌──┐ ┌──┐ ┌──┐              │
│  │  │ │  │ │  │              │
│  └──┘ └──┘ └──┘              │
```

**个人主页样式**：
- 在中间区域显示（替代视频区），右侧评论区隐藏
- 顶部有返回按钮，点击返回视频流
- 头像+昵称+ID+编辑资料按钮
- 关注/粉丝/获赞统计
- 作品/喜欢/收藏标签页
- 视频网格（3列）

#### Scenario: 查看个人主页
- **WHEN** 用户点击左侧导航栏"我的"
- **THEN** 中间区域显示个人主页，右侧评论区隐藏
- **AND** 点击返回按钮回到视频流

**关键改动**：

- 初始状态在视频区上方隐藏，显示时滑入视频区顶部

修改后：右侧固定面板，始终可见。

当前：底部5个导航项（首页/朋友/+/消息/我）。

## REMOVED Requirements

### Requirement: 底部导航栏
**Reason**: PC端抖音不使用底部导航栏，改用左侧导航栏
**Migration**: 所有底部导航功能迁移到左侧导航栏

### Requirement: 全屏视频滑动
**Reason**: PC端抖音视频不占满全屏，居中显示在中间区域
**Migration**: 视频改为居中播放器模式，上下切换替代滑动

### Requirement: 底部滑出评论面板
**Reason**: PC端抖音评论区固定在右侧，不需要滑出
**Migration**: 评论区改为右侧固定面板
