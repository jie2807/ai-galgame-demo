# 紫罗兰永恒花园首页美术升级 Spec (图片驱动)

## Why
当前页面布局正确，但整体美术效果仍然简陋，主要问题是使用 SVG 占位图标替代了实际的游戏图片资源。需要使用文生图技能生成高质量的艺术图片，替换所有占位元素，使页面达到设计稿级别的艺术品质。

## What Changes
- **使用文生图技能生成图片资源**
- 替换背景占位图为高质量艺术背景
- 生成角色头像图片
- 生成左侧菜单图标（人物、背包、信）
- 生成顶部货币图标（金币、宝石）
- 生成右侧功能按钮图标（羽毛笔、学校、收藏徽章）
- 生成活动横幅装饰图标
- 生成书本/书信装饰图片
- 生成底部横幅背景图片
- 更新 HTML/CSS 以使用生成的图片资源

## Impact
- 前端:首页美术效果大幅提升
- HTML 结构微调（SVG 占位替换为 `<img>` 标签）
- CSS 新增图片相关样式
- 新增图片资源目录

## ADDED Requirements

### Requirement: 背景图片
系统 SHALL 提供高质量的艺术背景图片:
- 紫罗兰永恒花园风格，暖色调室内场景
- 分辨率: 1920x1080 或更高
- 包含沙发、窗户、温暖光线的室内氛围
- 两位女性角色坐在沙发上的复古画面

#### 场景:背景展示
- **WHEN** 用户查看页面
- **THEN** 页面显示完整的艺术背景图

### Requirement: 角色头像
系统 SHALL 提供角色头像图片:
- 圆形裁剪的薇尔莉特·伊芙加登头像
- 风格:动漫/插画风格
- 分辨率: 至少 200x200

### Requirement: 菜单图标
系统 SHALL 提供左侧菜单图标:
- 人物图标:女性角色剪影
- 背包图标:复古手提箱
- 信图标:复古信封/信笺
- 风格:金色线条，复古优雅
- 分辨率: 至少 100x100

### Requirement: 货币图标
系统 SHALL 提供货币显示图标:
- 金币图标:复古金币
- 宝石图标:绿宝石/宝石
- 风格:3D 质感，金色/绿色
- 分辨率: 至少 80x80

### Requirement: 功能按钮图标
系统 SHALL 提供功能按钮图标:
- 羽毛笔图标:优雅羽毛笔
- 学校图标:复古学院建筑
- 收藏徽章:星形徽章
- 风格:金色线条，装饰性
- 分辨率: 至少 100x100

### Requirement: 装饰元素
系统 SHALL 提供:
- 书本/书信封面图片
- 活动横幅装饰（永恒花园、星空下）
- 分隔线和花纹装饰

## MODIFIED Requirements

### Requirement: 视觉风格
系统 SHALL 遵循设计稿的视觉风格:
- 整体色调:暖色系、复古风格、艺术感
- 所有图片资源保持风格统一
- 图片与 CSS 效果（光影、阴影）配合协调

## REMOVED Requirements
无
