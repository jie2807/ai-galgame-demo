# 用户视角功能优化 Checklist

## 签到系统
- [ ] 连续签到奖励机制正常工作，递增奖励表正确
- [ ] 签到日历视图显示当月日历，已签到日期金色标记，今日高亮
- [ ] 补签机制消耗宝石，1次/天限制，补签后连续天数恢复
- [ ] 转盘奖励池包含多种奖励类型，转盘动画流畅

## 抽卡系统
- [ ] 30抽保底机制正常工作，保底计数器显示正确
- [ ] 抽卡动画播放正常（单抽3秒，十连5秒，4星光效）
- [ ] 信件可通过多种途径获取（签到、章节、成就、邮箱、兑换）
- [ ] 重复角色自动转化为回忆碎片，兑换商店可正常兑换

## 邮箱系统
- [ ] 系统邮件在事件触发时自动发送，内容正确
- [ ] NPC书信在好感度达标时触发，内容与剧情相关
- [ ] 邮件奖励可正常领取，入账正确
- [ ] 邮箱图标显示未读红点数量

## 角色图鉴
- [ ] 角色图鉴以卡片网格展示，已解锁/未解锁状态正确
- [ ] 角色详情页显示完整信息（立绘、称号、简介、好感度、回忆录）
- [ ] 模型测试功能作为图鉴子功能正常工作
- [ ] 排序功能（好感度、获取时间、稀有度）正常

## 人偶养成学校
- [ ] 角色属性（表达力、共感力、观察力、书写力）可查看和提升

## 好感度系统

## 成就系统
- [ ] 成就进度条和数值显示正确
