# Task Dependencies
- Task 3 (邮箱) 依赖 Task 6 (好感度) — NPC书信需要好感度等级触发
- Task 5 (养成学校) 依赖 Task 6 (好感度) — 养成互动需要好感度系统支持
- Task 2 (抽卡) 的信件获取途径依赖 Task 1 (签到) 和 Task 8 (成就) — 签到和成就是信件来源
- Task 14 (数据持久化) 应最后实现 — 需要所有功能的数据结构稳定后再做导出导入
- Task 12 (新手引导) 应最后实现 — 需要所有功能完成后才能编写引导内容
- Task 11 (导航)、Task 13 (音效) 可与其他任务并行
