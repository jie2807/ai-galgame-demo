# 项目摘要（Taptap 商店上架用）

## 应用名称与描述
- **全称**：紫罗兰永恒花园
- **简称**：紫罗兰花园（PWA short_name）
- **当前副标题/描述**：AI 驱动的视觉小说互动游戏（manifest.webmanifest）
- **首页主题元素**：C.H 邮政公司、自动手记人偶、羽毛笔、书信、VIOLET 角色

## 核心玩法/功能
- AI 驱动的 Galgame 式视觉小说对话，支持打字机效果与角色立绘切换
- 官方剧本「末世方舟」：丧尸末日背景，含「废墟之弈」「钢铁之弈」「黎明之弈」三条路线
- 自定义剧本编辑器：玩家可创作世界观、NPC、地点、事件与开场场景
- Creator Wars（AI 工作台）：模拟 B站/微信/抖音/邮箱/股市等社交经营玩法
- 角色养成与抽卡系统、Live2D / 精灵立绘、书架/阅读器成书功能
- PWA 支持，移动端横屏适配，可添加到主屏

## 独特卖点
- AI 实时生成剧情，每次选择影响世界状态与多结局走向
- 官方剧本提供完整事件系统（时间线/阈值/随机/终局事件）
- 高自由度自定义剧本 + 社交模拟经营双模式
- 紫罗兰/书信/末世交织的沉浸式美术与叙事氛围

## 视觉风格关键词
- 暗紫、暖金、深褐、玻璃拟态（Liquid Glass）
- 羽毛笔、信封、蜡烛光、月光、胶片颗粒、扫描线
- 优雅衬线字体（Playfair Display / Noto Serif TC）

## 目标受众
- 视觉小说 / Galgame 玩家、AI 互动叙事爱好者、末日/科幻题材受众

## 当前状态
- 移动端上线适配已完成（PWA manifest、横屏锁定、触屏交互修复）
- 处于测试/试分发阶段，Taptap 上架准备中

## 版权/IP 风险
- **高风险**：应用名「紫罗兰永恒花园」与日本动漫《Violet Evergarden》（中译《紫罗兰永恒花园》）中文译名完全一致，角色 VIOLET、C.H 邮政公司、自动手记人偶等设定均与该 IP 高度重合，上架审核与版权投诉风险极大。
- **建议**：准备原创备选名称，如「紫罗兰书简」「末日书信」「Violet Letters」等，并移除/替换明显对应 IP 的视觉元素。
