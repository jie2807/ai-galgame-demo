# TapTap 上架宣传图制作 Spec

## Why
项目准备在 TapTap 上架测试版，目前缺乏用于商店详情页的宣传图。需要制作 3 张高质量横版宣传图，向玩家直观传达产品的核心玩法与卖点。

## What Changes
- 使用 `byted-seedream-image-generate` 生图技能，生成 3 张横版宣传图（1920×1080，landscape_16_9）
- 三张图分别覆盖三个主题：
  1. **主视觉图（KV）**：双世界氛围（末日废土 + 仙侠修真），Live2D 立绘角色环绕主体，呈现"AI 互动文字冒险"的沉浸感
  2. **核心玩法图**：展示自定义剧本、官方章节任务、好感度/抽卡/邮件/成就等系统矩阵
  3. **沉浸体验图**：还原视觉小说游玩场景——底部对话框 + 立绘 + 多分支多结局氛围
- 三张图统一风格、统一色调，构成一组完整素材
- **本次任务仅生成图片素材，不修改任何代码**
- 项目正式改名"随说"为后续独立任务，本任务中：
  - 宣传图**不得出现**"紫罗兰"字样或相关意象
  - App 名称可使用"随说"，或在图面中不出现具体 App 名（仅作视觉主图）
- **宣传图中不得包含**"AI 工作台"相关内容（该功能是否进入正式版未定）

## Impact
- Affected specs: 无（纯素材产出）
- Affected code: 无（不修改任何代码文件）
- 产出位置：`images/promo/` 目录（新建），文件命名 `promo-01-kv.png`、`promo-02-features.png`、`promo-03-experience.png`

## ADDED Requirements

### Requirement: TapTap 宣传图素材
系统 SHALL 产出 3 张符合 TapTap 商店详情页规格的横版宣传图，用于测试版上架。

#### Scenario: 主视觉图
- **WHEN** 玩家在 TapTap 商店详情页查看首张宣传图
- **THEN** 看到一张能体现"AI 互动文字冒险"核心定位的氛围图，画面融合末日废土与仙侠修真两种世界观，有 Live2D 立绘角色点缀
- **AND** 画面**不出现**"紫罗兰"字样或相关意象
- **AND** 画面**不出现**"AI 工作台"相关元素
- **AND** 图面整洁、构图聚焦，文字（若有）仅为简洁的标语或 App 名"随说"

#### Scenario: 核心玩法图
- **WHEN** 玩家查看第二张宣传图
- **THEN** 看到展示产品核心功能矩阵的图，至少涵盖：官方剧本（末日文/修仙文）、自定义剧本、章节任务、好感度、抽卡、邮件、成就等系统
- **AND** 各系统以图标 + 简短文案的形式排布，层次清晰
- **AND** **不含** AI 工作台相关功能
- **AND** **不含**"紫罗兰"字样

#### Scenario: 沉浸体验图
- **WHEN** 玩家查看第三张宣传图
- **THEN** 看到还原游戏内视觉小说游玩场景的图：横屏对话界面 + 角色立绘 + 多分支选项的氛围
- **THEN** 体现"多结局""高自由度""角色互动"的卖点
- **AND** **不含**"紫罗兰"字样、**不含** AI 工作台元素

#### Scenario: 素材规格
- **WHEN** 生图完成
- **THEN** 每张图为 1920×1080 分辨率（landscape_16_9）
- **AND** 三张图风格、色调统一，构成完整一组
- **AND** 文件保存至 `images/promo/` 目录，命名为 `promo-01-kv.png`、`promo-02-features.png`、`promo-03-experience.png`
