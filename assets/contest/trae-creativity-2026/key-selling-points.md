# 核心卖点提炼

> 以下卖点均基于项目现有文件（`index.html`、`data/characters.js`、`data/game-chapters.js`、`data/custom-script-prompts.js`、`data/npc-live2d-models.js`、`manifest.webmanifest` 等）提炼，并给出可验证的演示路径。

## 卖点 1：AI 实时生成的互动叙事体验

- **一句话定位**：玩家输入任意对话或行动，AI 即时续写剧情、生成 NPC 回应并动态调整世界状态。
- **功能支撑**：
  - `index.html` 中集成聊天引擎与 LLM 调用逻辑，支持 `say / act / think` 三种快捷输入模式。
  - 游戏中存在状态变量（`stateVars`），AI 会根据人口、食物、士气、关系值等变量变化推动剧情。
  - 支持上下文记忆，NPC 会根据当前局势给出不同回应。
- **可演示路径**：
  1. 打开首页 → 选择「官方剧本」
  2. 进入「末世方舟：废墟之弈」
  3. 在对话框中输入自定义指令（如「派小队去西边侦察」）或点击快捷回复
  4. 观察 AI 生成的 NPC 回复、旁白与状态条变化

## 卖点 2：完整官方剧本 + 状态驱动的多结局系统

- **一句话定位**：官方剧本并非线性小说，而是由时间线事件、阈值事件、随机事件与终局事件共同构成的策略叙事沙盘。
- **功能支撑**：
  - `data/game-chapters.js` 定义了「废墟之弈 / 钢铁之弈 / 黎明之弈」三章末世方舟剧本。
  - 每章包含 `timelineEvents`（按天数触发）、`thresholdEvents`（变量阈值触发）、随机事件与 `endingEvents`（多终局）。
  - 状态变量覆盖人口、食物库存、弹药、感染者威胁度、团结度、士气等维度。
- **可演示路径**：
  1. 首页 → 官方剧本 → 选择第一章「废墟之弈」
  2. 连续做出关键决策（如是否接纳新幸存者、是否向方舟求援）
  3. 观察顶部状态条与事件面板变化
  4. 在「凛冬降临」时根据累计状态触发不同结局

## 卖点 3：高自由度自定义剧本编辑器 + AI 一键生成

- **一句话定位**：从一句灵感出发，AI 自动生成完整世界观、角色、地点、事件与开场场景，玩家可立即进入自己创造的剧本。
- **功能支撑**：
  - `index.html` 中实现独立的「自定义篇章设定」编辑器（`customChapterEditor`）。
  - `data/custom-script-prompts.js` 提供基础 / 中级 / 高级三档生成提示词，控制生成内容的深度与复杂度。
  - 支持上传封面、添加标签、导入小说文件作为参考、自动生成场景背景与角色立绘开关。
- **可演示路径**：
  1. 首页 → 选择「自定义剧本」
  2. 点击「新建剧本」
  3. 在「智能生成」文本框中输入灵感，例如：「一个赛博朋克城市的赏金猎人冒险」
  4. 选择生成等级 → 点击生成 → 预览 JSON 设定 → 进入「游玩模式」开始体验

## 卖点 4：Live2D / TTS / BGM 三位一体的沉浸视听体验

- **一句话定位**：NPC 立绘、语音朗读与场景音乐会随剧情情绪与场景动态切换，强化视觉小说临场感。
- **功能支撑**：
  - `index.html` 引入 Live2D、PixiJS、Cubism Core 与本地 `live2d-renderer.js`，并配置 `data/npc-live2d-models.js` 的立绘映射。
  - 设置面板提供「Live2D 动态立绘」开关与「TTS 语音模型调配」，支持配置语音合成接口与自动播放。
  - `data/bgm-map.js` 与 `data/emotion-keywords.js` 提供场景音乐与情绪关键词映射。
- **可演示路径**：
  1. 进入任意剧本 → 点击设置 → 模型配置
  2. 配置 TTS provider 并开启自动播放
  3. 返回游戏，点击 NPC 对话旁的语音按钮，或等待自动朗读
  4. 切换场景 / 触发情绪标签，观察背景与立绘变化

## 卖点 5：PWA + 移动端横屏适配，随时随地可玩

- **一句话定位**：基于 Web 的跨端互动小说，支持添加到主屏、横屏锁定与触屏交互，桌面与移动设备均可完整体验。
- **功能支撑**：
  - `manifest.webmanifest` 配置 PWA 名称、图标、横屏显示与启动方式。
  - `index.html` 中通过 `screen.orientation.lock('landscape')` 锁定横屏，并设置 `viewport-fit=cover` 适配刘海屏。
  - 自定义剧本提供「纯文本模式」支持竖屏，兼顾移动端碎片化阅读。
- **可演示路径**：
  1. 使用手机浏览器打开项目
  2. 浏览器菜单选择「添加到主屏」
  3. 从主屏启动，观察启动画面与横屏锁定效果
  4. 在首页选择「自定义剧本 → 纯文本模式」，体验竖屏聊天式叙事

## 验证说明

- 以上卖点均能在当前代码与资源中找到对应实现，未依赖未完成的「AI工作台/Creator Wars」等占位功能。
- 演示时建议优先展示卖点 1、2、3，突出 AI 互动叙事与创作延展性；卖点 4、5 用于回答评审关于沉浸感与跨端能力的问题。
